## Key Features

- **Receipt Scanning:** Use device camera to scan receipts and documents
- **AI Categorization:** Automatic categorization using Apple Intelligence (100% on-device processing)
- **Spending Analytics:** Track spending trends and insights
- **PDF Export:** Generate professional tax reports and expense summaries
- **Return Tracking:** Never miss a return deadline with smart reminders
- **iCloud Sync:** Access receipts across all Apple devices (Pro feature)

## Privacy and Security

- **100% On-Device Processing:** All AI categorization happens on your device
- **No Data Sharing:** We never sell or share your data with third parties
- **Encrypted Storage:** All data encrypted at rest using iOS security features
- **Optional iCloud Sync:** Encrypted by Apple's CloudKit service
